# Proyecto Modularizado: Portfolio LFC

Este proyecto es el resultado de la modularización del portfolio original. Aquí encontrarás cada sección y funcionalidad separada en componentes y páginas para facilitar el mantenimiento y la escalabilidad.

## Estructura de Carpetas

- `components/`: Componentes reutilizables (Header, Sidebar, Footer, Botones, etc.)
- `pages/`: Páginas principales (`index`, `disenoweb`, `portfolio`, `contacto`)
- `styles/`: Estilos globales, por página y por componente
- `js/`: Scripts por página y por componente
- `assets/`: Imágenes, fuentes, etc.
- `locales/`: Archivos de traducción

## Cómo trabajar

- Modifica los componentes para cambiar partes reutilizables.
- Modifica las páginas para cambiar el contenido principal de cada sección.
- Los estilos están divididos para evitar conflictos entre páginas/componentes.

## Compatibilidad

El proyecto mantiene compatibilidad con el código original, pero con una estructura mucho más mantenible y escalable.
